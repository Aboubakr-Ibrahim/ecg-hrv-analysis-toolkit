# Generated results

Store only reviewed, permission-safe CSV summaries and quality-control figures here. Generated outputs are ignored by default to prevent accidental publication of private or unreviewed material.

## Required review before publication

- Confirm record ownership and publication permission
- Review every detected R peak and major artifact region
- Report retained/rejected interval counts and retained fraction
- Confirm successive-pair count for RMSSD, pNN50, SD1, and SD2
- Confirm frequency-domain eligibility and analyzed duration
- Document channel, sampling rate, protocol, and rhythm limitations
- Remove filenames or metadata that could identify a person or client

## Current status

The Version 2 toolkit has passed static review. No clinical conclusion, group comparison, environmental association, or reproduced performance result is claimed until the current code is run on documented, permission-safe recordings and reviewed.
